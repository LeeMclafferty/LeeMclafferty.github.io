Software Renderer

This software renderer is my current passion project, which I'm working on alongside my school coursework. Its primary objective is to provide me with a comprehensive introduction to computer graphics and linear algebra by manually implementing essential components wherever feasible.

To accomplish this, I am leveraging the SDL library for window creation, the final display through the SDL renderer, and handling user input. However, everything else is being built from scratch, with a strong focus on adhering to modern C++ best practices.

At the core of my renderer is a single "engine" instance, serving as the central entry point for the program. This engine efficiently manages crucial elements like the renderer itself, the window, the scene, and other important core classes.

Currently, the program supports various features, including reading .obj files, mesh transformations, multiple display methods, and backface culling. My next steps involve implementing face rendering order with a Z-Buffer, adding textures, and incorporating lighting techniques to enhance realism.

By the end of this project, my goal is to have a profound understanding of computer graphics principles, preparing me to delve into OpenGL, which can abstract much of the manual implementation for me. With this solid foundation, I look forward to exploring even more complex and exciting graphics projects in the future.
